## C. Ejercicios Prácticos

### Nivel Básico
1. **Calculadora Científica**
   - Crea una calculadora que convierta entre unidades (Celsius-Fahrenheit, gramos-moles, etc.)
   - Implementa funciones para cálculos científicos básicos (pH, concentraciones, diluciones)

2. **Análisis de Datos Climáticos**
   - Descarga datos de temperatura históricos
   - Calcula promedios, máximos y mínimos por mes
   - Visualiza tendencias temporales

3. **Simulación de Crecimiento Bacteriano**
   - Implementa el modelo de crecimiento logístico
   - Simula diferentes condiciones (temperatura, nutrientes)
   - Visualiza curvas de crecimiento

### Nivel Intermedio
1. **Clasificador de Flores Iris**
   - Implementa regresión logística desde cero
   - Evalúa el modelo con diferentes métricas
   - Visualiza fronteras de decisión

2. **Análisis de Espectros**
   - Carga datos espectroscópicos reales
   - Encuentra picos y calcula áreas bajo la curva
   - Normaliza y compara múltiples espectros

3. **Optimización de Experimentos**
   - Implementa búsqueda de parámetros óptimos
   - Usa validación cruzada para evaluar modelos
   - Visualiza superficies de respuesta

### Nivel Avanzado
1. **Red Neuronal para Predicción**
   - Implementa una red neuronal desde cero
   - Entrena con datos científicos reales
   - Evalúa sobreajuste y regularización

2. **Análisis de Series Temporales**
   - Implementa modelos ARIMA
   - Predice tendencias futuras
   - Evalúa precisión de predicciones

3. **Procesamiento de Imágenes Científicas**
   - Carga y procesa imágenes microscópicas
   - Implementa segmentación y detección de objetos
   - Cuantifica características morfológicas

## D. Glosario de Términos

### Términos de Programación
| Término | Definición |
|---------|------------|
| Algoritmo | Conjunto de instrucciones para resolver un problema |
| Array | Estructura de datos para almacenar elementos del mismo tipo |
| Bug | Error en un programa de computadora |
| Compilador | Programa que traduce código fuente a código máquina |
| Depuración | Proceso de encontrar y corregir errores en el código |
| Función | Bloque de código reutilizable que realiza una tarea específica |
| Iteración | Repetición de un proceso o conjunto de instrucciones |
| Librería | Colección de funciones y módulos reutilizables |
| Loop | Estructura de control que repite un bloque de código |
| Variable | Contenedor para almacenar datos en memoria |

### Términos de Ciencia de Datos
| Término | Definición |
|---------|------------|
| Dataset | Conjunto de datos organizados para análisis |
| Feature | Variable de entrada en un modelo de machine learning |
| Label | Variable de salida o objetivo en un modelo |
| Overfitting | Cuando un modelo se ajusta demasiado a los datos de entrenamiento |
| Underfitting | Cuando un modelo es demasiado simple para capturar patrones |
| Validation | Proceso de evaluar un modelo con datos no vistos |
| Cross-validation | Técnica para evaluar modelos dividiendo datos múltiples veces |
| Hyperparameter | Parámetro configurable de un modelo de machine learning |
| Pipeline | Secuencia de pasos para procesar datos y entrenar modelos |

### Términos de Estadística
| Término | Definición |
|---------|------------|
| Media | Promedio de un conjunto de valores |
| Mediana | Valor que divide un conjunto ordenado en dos partes iguales |
| Moda | Valor que aparece con mayor frecuencia |
| Desviación estándar | Medida de dispersión de los datos |
| Varianza | Cuadrado de la desviación estándar |
| Correlación | Medida de relación entre dos variables |
| Regresión | Técnica para modelar la relación entre variables |
| Hipótesis | Suposición que se prueba mediante análisis estadístico |
| p-value | Probabilidad de obtener resultados al menos tan extremos como los observados |

### Términos de Machine Learning
| Término | Definición |
|---------|------------|
| Supervised Learning | Aprendizaje con datos etiquetados |
| Unsupervised Learning | Aprendizaje sin datos etiquetados |
| Classification | Tarea de predecir categorías discretas |
| Regression | Tarea de predecir valores continuos |
| Clustering | Agrupar datos similares sin etiquetas previas |
| Neural Network | Modelo inspirado en el cerebro biológico |
| Deep Learning | Redes neuronales con múltiples capas |
| Gradient Descent | Algoritmo de optimización para entrenar modelos |
| Backpropagation | Algoritmo para calcular gradientes en redes neuronales |
