## 🛠️ **Configuración del Entorno Científico**

### **Requisitos Mínimos**
- Computadora con Windows, macOS o Linux
- Conexión a internet para instalaciones
- 4GB de RAM (8GB recomendado)
- 2GB de espacio libre

### **Instalación Recomendada**
```bash
# 1. Instalar Python 3.9 o superior
# Descargar desde: https://www.python.org/downloads/

# 2. Verificar instalación
python --version

# 3. Instalar Jupyter Notebook (nuestro laboratorio digital)
pip install notebook

# 4. Instalar librerías científicas básicas
pip install numpy pandas matplotlib seaborn scipy

# 5. Instalar librerías de IA (opcional para ahora)
pip install scikit-learn tensorflow
```

### **Entorno de Desarrollo Sugerido**
- **Editor de código**: VS Code (gratuito y potente)

  - Python (Microsoft)
